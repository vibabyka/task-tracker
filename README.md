#  Трекер задач

Канбан-доска для управления задачами с локальным хранением данных и красивым розовым дизайном.

#  Возможности

- ✅ Три колонки: «Новые», «В работе», «Готово»
- ✅ Создание, редактирование, удаление задач
- ✅ Drag & Drop между колонками (HTML5 API)
- ✅ Сохранение в localStorage
- ✅ Приоритеты с цветовыми метками
- ✅ Счётчики задач в каждой колонке
- ✅ Адаптивный дизайн (360px–1920px)
- Тёмная/светлая тема с переключением иконок
- Экспорт/импорт задач в JSON
- Розово-фиолетовый градиентный дизайн

#  Технологии

- HTML5 (семантическая разметка)
- CSS3 (Flexbox, CSS-переменные, медиа-запросы, градиенты, анимации)
- JavaScript ES6+ (vanilla JS, DOM API, Drag & Drop API, localStorage)
